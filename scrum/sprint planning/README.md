# Sprint Planning – Projeto Avalia+

Esta pasta contém os registros das **Sprint Plannings** realizadas pela equipe do projeto Avalia+.

A Sprint Planning é a reunião onde a equipe define:

- quais tarefas serão realizadas na sprint
- quais funcionalidades serão desenvolvidas
- como o trabalho será dividido entre os integrantes

---

## Objetivo da Sprint Planning

A Sprint Planning tem como objetivo organizar o trabalho da equipe para cada sprint, definindo as tarefas que deverão ser realizadas durante o período de desenvolvimento.
