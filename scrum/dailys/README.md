# Daily Scrum – Projeto Avalia+

Esta pasta contém os registros das reuniões de acompanhamento (Daily Scrum) realizadas durante o desenvolvimento do projeto **Avalia+**.

As dailys são realizadas duas vezes por semana e têm como objetivo acompanhar o andamento das tarefas da equipe, identificar possíveis dificuldades e alinhar os próximos passos da sprint.

Cada documento de daily contém:

- Data da reunião
- Participantes
- Atualização das atividades de cada integrante
- Impedimentos ou dificuldades encontradas
- Alinhamentos realizados durante a reunião
- Próximos passos da equipe

Esses registros permitem acompanhar a evolução do projeto ao longo das sprints e garantir a organização e comunicação entre os membros da equipe.
