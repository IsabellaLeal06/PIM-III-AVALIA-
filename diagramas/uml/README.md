# Diagramas UML – Projeto Avalia+

Esta pasta contém os diagramas UML do projeto **Avalia+**, utilizados para representar a estrutura e o comportamento do sistema.

Os diagramas auxiliam na compreensão das funcionalidades, da arquitetura e das interações entre os elementos do sistema.

---

## Diagramas presentes

- **Diagrama de Casos de Uso**  
  Representa as interações entre os usuários e o sistema, evidenciando as principais funcionalidades disponíveis na plataforma.

- **Diagrama de Classes**  
  Apresenta a estrutura do sistema, incluindo classes, atributos, métodos e seus relacionamentos.

- **Diagrama de Sequência**  
  Demonstra o fluxo de interação entre os componentes do sistema ao longo do tempo, detalhando a execução de funcionalidades específicas.

---

## Objetivo dos Diagramas

Os diagramas têm como objetivo:

- Representar os requisitos funcionais do sistema  
- Auxiliar na modelagem e organização da arquitetura  
- Facilitar o entendimento das funcionalidades e fluxos do sistema  
- Apoiar o planejamento e desenvolvimento da aplicação  

---

Esses diagramas são fundamentais para a modelagem do sistema Avalia+, servindo como base para o desenvolvimento e implementação da solução.
