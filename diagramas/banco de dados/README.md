# Banco de Dados – Projeto Avalia+

Esta pasta contém os diagramas relacionados à modelagem do banco de dados do sistema Avalia+.

Os modelos foram desenvolvidos com o objetivo de estruturar, organizar e representar os dados necessários para o funcionamento da plataforma.

---

## Conteúdo da pasta

- mer
- der

---

## Modelo Entidade-Relacionamento (MER)

O MER representa a visão conceitual do banco de dados.

Neste modelo são apresentadas as principais entidades do sistema, seus atributos essenciais e os relacionamentos entre elas, sem foco em detalhes técnicos.

Seu objetivo é facilitar a compreensão da estrutura geral do sistema.

---

## Diagrama Entidade-Relacionamento (DER)

O DER representa a visão lógica do banco de dados.

Neste diagrama são detalhados os atributos das entidades, bem como as chaves primárias (PK), chaves estrangeiras (FK) e as cardinalidades dos relacionamentos.

O DER serve como base para a implementação do banco de dados.

---

## Objetivo

Garantir uma modelagem de dados clara, organizada e alinhada com os requisitos do sistema, permitindo uma implementação consistente e eficiente do banco de dados.
